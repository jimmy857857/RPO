# 记账+聚会游戏微信小程序

一款融合「轻量化记账管理」与「即时性聚会游戏」的双场景微信小程序，主打"日常管账+聚会破冰"的复合型需求。

## 功能特色

### 记账管理系统
- **快速记账**: 3秒快速记账，支持收入/支出切换
- **智能分类**: 预设15+高频分类，支持AI语音记账
- **数据统计**: 可视化图表展示收支趋势和分类占比
- **预算管理**: 月度预算设置，超额预警提醒
- **数据同步**: 基于微信云开发的多设备数据同步

### 聚会游戏系统
- **转盘游戏**: 内置多种模板，支持自定义选项
- **真心话大冒险**: 200+题目库，支持自定义题目
- **骰子游戏**: 单人/多人模式，支持对战功能
- **即时互动**: 零学习成本，解决聚会"冷场"问题

## 技术架构

- **前端**: 微信小程序原生框架 + Vant Weapp组件库
- **后端**: 微信云开发（云函数 + 云数据库 + 云存储）
- **动画**: CSS3动画 + Canvas 2D渲染
- **数据可视化**: 微信小程序图表库

## 快速开始

### 环境要求
- 微信开发者工具最新版
- 微信小程序 AppID
- 开通微信云开发服务

### 开发步骤
1. 克隆项目到本地
2. 在微信开发者工具中导入项目
3. 配置云开发环境
4. 上传云函数
5. 编译运行

### 云开发配置
1. 在 `app.js` 中配置云环境 ID
2. 在云开发控制台创建以下集合：
   - `records`: 记账记录
   - `budgets`: 预算设置
   - `questions`: 游戏题库
3. 设置数据库安全规则

## 核心功能实现

### 记账功能
- 支持快速记账和详细记账两种模式
- 自动分类识别和手动分类选择
- 数据实时同步和本地缓存

### 游戏功能
- 转盘游戏支持自定义模板和颜色
- 真心话大冒险支持题目管理和随机抽取
- 骰子游戏支持单人和多人对战模式

### 数据统计
- 饼图展示支出分类占比
- 柱状图展示收支趋势
- 支持按日、周、月、年筛选数据
