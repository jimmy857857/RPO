# 记账游戏助手 (RPO)

一个集记账管理和聚会游戏于一体的微信小程序，采用现代化的 UI 设计，提供流畅的用户体验。

## ✨ 项目特色

- 🎨 **现代化 UI 设计**：采用苹果新系统风格的液态玻璃（Liquid Glass）效果
- 🌈 **动态渐变背景**：紫→粉→蓝→青的流动渐变效果，视觉体验更佳
- 💰 **智能记账管理**：支持支出/收入记录、分类管理、预算设置、数据统计
- 🎮 **聚会游戏**：转盘游戏、真心话、骰子游戏，让聚会更有趣
- 📱 **响应式布局**：优化的一屏显示，交互更友好
- ☁️ **云开发支持**：基于微信云开发，数据安全可靠

## 📋 功能模块

### 记账管理

- **快速记账**：支持支出/收入记录，多种分类选择
- **数据统计**：可视化展示收支情况，支持多维度分析
- **预算设置**：月度预算管理，实时跟踪预算执行情况
- **最近记录**：快速查看最近的记账记录

### 聚会游戏

- **转盘游戏**：自定义转盘选项，支持多种模板
- **真心话**：经典聚会游戏，增进朋友间的了解
- **骰子游戏**：随机骰子游戏，增加聚会趣味性

### 个人中心

- **用户设置**：个人信息管理
- **工具集**：实用工具集合
- **反馈建议**：用户反馈渠道

## 🛠️ 技术栈

- **框架**：微信小程序原生开发
- **云服务**：微信云开发
- **UI 组件**：@vant/weapp
- **构建工具**：Gulp
- **代码规范**：ESLint

## 🚀 快速开始

### 环境要求

- Node.js >= 12.0.0
- 微信开发者工具
- 微信小程序账号（已开通云开发）

### 安装步骤

1. **克隆项目**

   ```bash
   git clone https://github.com/jimmy857857/RPO.git
   cd RPO
   ```

2. **安装依赖**

   ```bash
   npm install
   ```

3. **配置云开发**

   - 在微信开发者工具中打开项目
   - 开通云开发服务
   - 配置云开发环境 ID

4. **构建项目**

   ```bash
   npm run dev
   ```

5. **在微信开发者工具中预览**

   - 打开微信开发者工具
   - 导入项目目录
   - 选择对应的 AppID 或使用测试号
   - 点击编译预览

## 📝 开发命令

```bash
# 开发模式（编译 + 预览）
npm run dev

# 仅编译
npm run compile

# 代码检查
npm run lint
```

## 🎨 UI 设计特点

### 液态玻璃效果

- 使用 `backdrop-filter: blur()` 实现毛玻璃模糊效果
- 半透明背景配合高饱和度，营造现代感
- 多层阴影和光效，增强视觉层次

### 动态渐变背景

- 5 色渐变（紫→粉→蓝→青）
- 15 秒循环动画，背景持续流动
- 统一应用到所有页面，保持视觉一致性

### 交互优化

- 流畅的动画过渡（使用 cubic-bezier 弹簧曲线）
- 点击反馈动画（缩放、位移、扫光效果）
- 选中状态高亮显示

## 📱 页面说明

### 首页

- 显示本月预算信息（已支出、剩余、总预算）
- 预算进度条可视化
- 快速访问记账、统计、预算设置
- 聚会游戏入口
- 最近记账记录

### 记账页面

- 支出/收入类型切换
- 金额输入（支持快捷金额选择）
- 分类选择（根据类型动态过滤）
- 日期、支付方式、备注设置
- 一屏显示，交互友好

### 游戏页面

- **转盘游戏**：自定义转盘选项和颜色
- **真心话**：经典问答游戏
- **骰子游戏**：随机骰子

### 统计页面

- 收支数据可视化
- 分类统计
- 时间维度分析

### 预算页面

- 月度预算设置
- 预算执行情况跟踪

## 🔧 配置说明

### 云开发配置

项目使用微信云开发，需要配置以下集合：

- `records`：记账记录
- `budgets`：预算设置

### 自定义 TabBar

项目使用自定义 TabBar，需要在 `app.json` 中设置 `"custom": true`。

### 环境变量

云开发环境使用动态环境配置，无需手动设置环境 ID。
