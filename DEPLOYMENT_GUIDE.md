# 记账+聚会游戏微信小程序部署指南

## 项目概述
本项目是一个融合记账管理和聚会游戏的微信小程序，基于微信云开发架构。

## 环境要求
- 微信开发者工具（最新版本）
- 微信小程序 AppID
- Node.js 14+

## 部署步骤

### 1. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目根目录
4. 输入您的 AppID（或使用测试号）

### 2. 云环境配置
1. 在开发者工具中点击"云开发"
2. 创建新的云环境或使用现有环境
3. 在 `miniprogram/app.js` 中配置环境 ID

### 3. 云函数部署
1. 右键点击 `cloudfunctions/quickstartFunctions` 文件夹
2. 选择"上传并部署：云端安装依赖"
3. 等待部署完成

### 4. 数据库初始化
项目会自动创建以下集合：
- users（用户信息）
- records（记账记录）
- budgets（预算设置）
- games（游戏数据）
- backups（备份记录）

### 5. 本地测试
1. 点击"编译"按钮
2. 在模拟器中测试功能
3. 使用真机预览功能进行实际测试

## 功能模块说明

### 记账管理系统
- **快速记账**：首页"记一笔"按钮，支持收入/支出分类
- **数据统计**：饼图展示支出占比，柱状图展示趋势
- **预算管理**：设置月度预算，超额预警

### 聚会游戏系统
- **转盘游戏**：内置多种模板，支持自定义
- **真心话大冒险**：随机抽题，支持自定义题目
- **骰子游戏**：模拟实体骰子功能

### 个人中心
- 主题设置、数据导出、实用工具等

## 开发注意事项
1. 所有用户数据存储在云数据库，确保数据安全
2. 使用微信内容安全 API 过滤敏感内容
3. 遵循微信小程序设计规范
4. 定期备份重要数据

## 上线准备
1. 完成小程序注册和备案
2. 审核内置题库和图片资源
3. 进行功能测试和性能测试
4. 提交微信审核
